# BondedDonations


### Requires truffle5: 
    ```
    npm install -g truffle@beta
    ```
    

To run: 

1. Start ganache with block times: 
    ```
    ganache-cli -b 3
    ```

2. Compile and migrate contracts:
    ```
    truffle console
    compile
    migrate
    ```
3. Run node: 
    ```
    npm run start
    ```
